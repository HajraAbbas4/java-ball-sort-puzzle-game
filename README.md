# Java Ball Sort Puzzle Game

This project is a Java-based puzzle game developed using Java Swing. The game is inspired by ball sorting puzzle games, where players move colored balls between tubes and try to arrange balls of the same color together.

The project demonstrates the use of Java GUI programming, Stack data structure, event handling, and basic game logic.

## Features
- Interactive graphical user interface
- Colorful ball sorting puzzle
- Mouse click-based gameplay
- Stack-based tube management
- Win condition checking
- Simple game window using Java Swing

## Technologies Used
- Java
- Java Swing
- AWT
- Stack Data Structure

## Concepts Covered
- Object-Oriented Programming
- GUI Development
- Event Handling
- Stack Operations
- Game Logic
- Data Structures

## How It Works
1. The game creates multiple tubes with colored balls.
2. Balls are stored using Stack data structures.
3. The player selects one tube and moves the top ball to another tube.
4. The goal is to arrange each tube with balls of the same color.
5. When all balls are sorted correctly, a winning message is displayed.
